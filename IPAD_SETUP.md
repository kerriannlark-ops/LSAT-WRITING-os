# iPad Refresh and Install

Use these steps after uploading a new `index.html`, `manifest.webmanifest`, or `service-worker.js` build.

## If the site is already installed on Home Screen

1. Open the live site URL in Safari.
2. Refresh once.
3. Wait a few seconds so the new service worker can install.
4. Close the Home Screen app if it is open.
5. Reopen the Home Screen app.

If the old version still appears:

1. Delete the Home Screen app.
2. In Safari, open the live site URL again.
3. Refresh once.
4. Tap `Share -> Add to Home Screen`.

## If you are installing for the first time

1. Open the live site URL in Safari.
2. Tap `Share`.
3. Tap `Add to Home Screen`.
4. Launch the installed app from the Home Screen.
