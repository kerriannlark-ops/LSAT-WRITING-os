## Core Curriculum Philosophy
The upgraded app should teach the LSAT essay as a **controlled reasoning system**, not a generic essay.

### Non-negotiable reasoning chain
Every body paragraph should train this chain:
1. **Claim** - what you are proving
2. **Mechanism** - how it becomes true
3. **Evidence/Perspective** - what in the prompt supports it
4. **Inference** - what the evidence means
5. **Counterpoint** - strongest objection
6. **Limitation/Rebuttal** - why the objection fails under the right conditions
7. **Comparative conclusion** - why your side is stronger on the chosen standard

### Non-negotiable strategic habits
The app should repeatedly enforce:
- choose a side quickly
- define the standard for “best” / “stronger” / “more persuasive”
- synthesize perspectives topic-by-topic
- avoid source-by-source summary
- explain mechanism and inference explicitly
- use clear comparative language
- write short, controlled sentences under pressure

## Module 1 - Prompt Deconstruction + Scope Box
### Goal
Train the user to narrow the prompt before drafting.

### Core instruction
Teach this sequence:
1. identify ask-type
2. underline operative verbs
3. spot constraints
4. identify evaluation standard
5. build a scope box

### Final scope-box formula
Use this everywhere:
- **Topic subset | population/context | timeframe/condition | analytical lens | comparison rule**

### Scope-box drill sequence
1. **Ask-Type ID Drill**
   - evaluate
   - advocate
   - compare
   - diagnose
   - design
   - forecast
2. **Constraint Capture Drill**
   - temporal window
   - institutional setting
   - population subset
   - condition / exception
3. **Scope Box Builder**
   - user completes all 5 parts
4. **Exclusion Drill**
   - identify 2 ideas that are outside the scope box and should be excluded

### Coach rules
Flag if the user:
- writes too broad a topic
- forgets the comparison rule
- has no analytical lens
- treats the issue as a vague opinion instead of an evaluative task

---

## Module 2 - Perspective Extraction + Stakeholder Lattice
### Goal
Move the user from passive reading to structured ideological mapping.

### Core instruction
For each perspective, the user should extract:
- core claim
- assumption
- strength
- weakness
- what value it prioritizes
- whether it supports, complicates, or challenges the user’s thesis

### Add a stakeholder lattice drill
Prompt the user to identify:
- author / advocate
- critic
- affected population
- institutional actor
- long-term consequences / civic lens

### Best drill sequence
1. **Perspective Audit (6 minutes)**
2. **Common Points Grid**
   - where perspectives overlap
   - where they contradict
3. **Topic-by-Topic vs Source-by-Source Sorting Drill**
   - show a weak source-by-source summary
   - require user to convert it into topic-by-topic synthesis

### Coach rules
Flag if the draft:
- name-drops perspectives without analysis
- uses too many perspectives shallowly
- summarizes in source order
- fails to connect perspectives to the thesis

---

## Module 3 - Mechanism Lab
### Goal
Fix the single biggest weakness across the documents: weak causal explanation.

### Core teaching line
**A mechanism sentence explains how one condition produces the result you are claiming.**

### Mechanism question set
Force the user to answer:
- What exactly does this give the actor?
- What does that allow, prevent, reduce, or preserve?
- Why does that lead to the claimed result?
- What is the full causal chain?

### Drill set
1. **3-Step Mechanism Builder**
   - claim -> what it gives -> what that allows -> final result
2. **Verb Precision Drill**
   Replace vague verbs with causal verbs such as:
   - enables
   - reduces
   - preserves
   - prevents
   - accelerates
   - signals
   - broadens
   - narrows
   - transfers
   - compounds
3. **Mechanism Expansion Drill**
   Turn one vague sentence into 3 precise sentences.
4. **Concrete Mechanism Drill**
   Require one concrete mechanism from this list when relevant:
   - internships
   - capstones
   - employer partnerships
   - writing-intensive coursework
   - quantitative literacy
   - professional communication
   - institutional signaling
   - analytical transfer

### Coach rules
Flag drafts that use:
- “this helps” without saying how
- “this is important” without mechanism
- abstract nouns without causal explanation

---

## Module 4 - Evidence -> Inference -> Why It Matters
### Goal
Stop paragraphs from dying after evidence.

### Core teaching line
**Evidence alone does not score. Evidence must be interpreted.**

### Required sentence chain
- Evidence/perspective sentence
- Inference sentence
- Why-it-matters sentence

### Drill set
1. **This Matters Because Drill**
   Add a why-it-matters sentence to every evidence sentence.
2. **Inference Ladder Drill**
   - evidence
   - inference
   - broader consequence
3. **Weak vs Strong Inference Drill**
   Show surface-level explanation and force deeper reasoning.
4. **Perspective-to-Meaning Drill**
   User must explain what the perspective proves, not just restate it.

### Coach rules
Flag when paragraphs do:
- evidence -> claim only
- quote-like summary with no interpretation
- “this shows” with no stated consequence

---

## Module 5 - Thesis + Standard + Introduction
### Goal
Make the thesis specific, comparative, and scoped.

### Core teaching line
A good LSAT thesis needs:
- a side
- a standard
- a hedge/qualification if needed
- a reason preview
- alignment with the scope box

### Thesis formula set
1. **Direct comparative thesis**
   - While X has force, Y better serves [population/context] because [criterion 1] and [criterion 2].
2. **Qualified thesis**
   - Under these conditions, Y is more persuasive because...
3. **Integrated thesis**
   - The stronger position is not absolute rejection of X, but a narrower model in which...

### Intro formula
1. narrow issue using scope box
2. state the central tension
3. announce the thesis and governing standard

### Drill set
1. **Thesis Precision Drill**
   Rewrite one thesis in 3 versions:
   - stronger
   - more nuanced
   - more aggressive
2. **Standard Definition Drill**
   Define what “best serves” means in measurable terms.
3. **Intro Compression Drill**
   Build an intro in 3 sentences max.

### Coach rules
Flag if the thesis:
- is too vague
- sounds neutral instead of comparative
- lacks a metric
- ignores scope constraints

---

## Module 6 - Counterargument + Limitation Studio
### Goal
Turn weak objections into real rebuttal logic.

### Core teaching line
The counter paragraph is not a throwaway concession; it is where the writer proves comparative control.

### Counter sequence
1. acknowledge the strongest objection fairly
2. articulate the real merit in that objection
3. expose the hidden assumption
4. narrow it with a limiting condition
5. compare it back to your thesis

### Drill set
1. **Weak Counter -> Strong Counter Upgrade**
2. **Assumption Exposure Drill**
   - what must be true for the objection to succeed?
3. **Limitation Clause Drill**
   sentence stems:
   - however, this assumes...
   - this concern is strongest only when...
   - that objection weakens once we recognize...
4. **Counter Precision Drill**
   write 2 counters + 1 real limitation for each

### Coach rules
Flag if the rebuttal:
- attacks a strawman
- repeats the thesis without narrowing the objection
- lacks a limiting condition
- never returns to comparison

---

## Module 7 - Paragraph Skeleton Mastery
### Goal
Make every body paragraph complete, not half-built.

### Required paragraph skeleton
- claim
- mechanism
- evidence/perspective
- inference
- counterpoint
- limitation
- comparative conclusion

### Drill set
1. **Full Skeleton Drill (6-8 sentences max)**
2. **Paragraph 2 Expansion Drill**
   specifically for the common weak second paragraph problem
3. **Comparative Ending Drill**
   replace weak endings (“this is better”) with criterion-based comparative endings
4. **Timed Band-6 Paragraph Drill (12-15 minutes)**
   build one paragraph that can stand alone

### Coach rules
Flag if a paragraph is missing:
- mechanism
- inference
- limitation
- comparative weigh language

---

## Module 8 - Outline + Essay Architecture
### Goal
Move from paragraph skill to full essay control.

### Best essay structure from the sources
- **Paragraph 1:** intro / thesis / standard
- **Paragraph 2:** first affirmative argument with perspective integration
- **Paragraph 3:** second affirmative argument with another perspective or distinct dimension
- **Paragraph 4:** tactical rebuttal / counter + limitation
- **Paragraph 5:** concise conclusion

### Critical architecture rule
Train **topic-by-topic** synthesis, not **source-by-source** sequence.

### Outline drill set
1. **Strategic Outline Template**
   - thesis
   - reason 1
   - reason 2
   - counter
   - conclusion
2. **Perspective Allocation Drill**
   decide where each perspective belongs
3. **Order Justification Drill**
   explain why paragraph order is strongest

### Coach rules
Flag if the outline:
- mirrors prompt order
- overuses all 4 perspectives shallowly
- lacks a dedicated rebuttal section

## Module 9 - Timed Performance System
### Goal
Use the curriculum under pressure.

### Official performance mode
- 15 min prewriting
- 35 min writing

### Training modes
- official 15/35
- extended training 23/53
- focused microdrills (10/15/20 min)

### Best prewriting checklist
1. gut reaction
2. perspective audit
3. scope box
4. thesis anchor
5. 2 reasons
6. strongest counter
7. comparison rule

### Best writing checklist
1. intro written fast
2. body 1 full chain
3. body 2 full chain
4. counter paragraph with real limitation
5. conclusion with comparative restatement
6. 3-minute clean-up

### Drill set
1. **Perspective Extraction (6 mins)**
2. **Full Skeleton Paragraph (10 mins)**
3. **Counter Precision Drill (8 mins)**
4. **Timed High-Scoring Paragraph (15 mins)**
5. **Full Timed Essay**
6. **Post-Timed Rewrite of Weakest Paragraph**

### Coach rules
Track:
- prewriting minutes used
- paragraph completion count
- word count by phase
- coach category weaknesses
- whether the counter paragraph was actually finished
